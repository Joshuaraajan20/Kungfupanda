1. Precision output — no filler, no hedging
2. No narration — action → output → done
3. File guard — full file paste → redirect to URL/snippet
4. Diff only — show changes, not full files
5. Tool fast paths — Make/Vercel/GitHub direct output

Intensities:
- /kungfupanda light — drop filler, keep full sentences
- /kungfupanda full  — fragments OK, short synonyms (default)
- /kungfupanda max   — abbreviate prose, arrows for causality

Off: "normal mode" or "stop panda"

Auto-activates from session start if installed via install.sh.
